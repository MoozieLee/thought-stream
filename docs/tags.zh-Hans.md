# 标签

ThoughtStream 将内联标签视为捕获时的快捷方式，而非长期的数据权威来源。

## 支持的标签格式

ThoughtStream 支持单词内联标签，例如：

- `#work`
- `#thoughtstream`
- `#code-review`
- `#weekly_review`

标签中不能包含空格。

多词概念推荐使用：

- kebab-case（连字符）
- snake_case（下划线）

## 捕获语义

`add` 操作时：

- 内联 `#tag` 标记会自动提取到结构化的 `tags` 字段中
- 提取后的标签仍保留在原始 `content` 中

`update --content` 操作时：

- 内联 `#tag` 标记会重新解析
- 新检测到的标签会被添加
- 已有标签不会自动移除

这意味着内联标签有助于快速捕获，但后续的标签管理不依赖重写原始笔记文本。

## 示例

输入：

```text
干完现在的活 #工作
```

存储为：

- `content`：`干完现在的活 #工作`
- `tags`：`["工作"]`

输入：

```text
#生活 买一把香蕉
```

存储为：

- `content`：`#生活 买一把香蕉`
- `tags`：`["生活"]`

## 设计理念

本项目倾向：

- 低摩擦的捕获体验
- 简洁的内联标签语法
- 捕获后使用结构化元数据

刻意避免了在捕获过程中强制用户管理标签。
