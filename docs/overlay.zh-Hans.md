# 覆盖层指南

## 覆盖层是做什么的

覆盖层是 ThoughtStream 里的主要捕获界面。

它适合用来：

- 快速记下一条想法
- 不打开完整工作区也能做轻量搜索和回顾
- 把旧笔记复用成新草稿的起点

## 如何打开覆盖层

使用：

```text
Shift + Command + Space
```

面板会从屏幕中央弹出，并立即把焦点放到输入框。

## 斜杠命令

覆盖层目前支持这些命令：

- `/tail`
  - 显示最近的笔记
- `/tail 20`
  - 显示指定数量的最近笔记
- `/search <query>`
  - 全文搜索
- `/today`
  - 查看今天的笔记
- `/tag <tag>`
  - 按单个标签筛选
- `/archive`
  - 查看已归档笔记
- `/keys`
  - 查看快捷键参考
- `/hide`
  - 收起当前结果面板
- `/help`
  - 查看命令列表
- `/exit`
  - 关闭面板

说明：

- `/tag` 只接受一个 token，例如 `/tag work`
- `/search` 必须带查询词
- `/tail` 可以不带数字，也可以带一个正整数

## 键盘快捷键

输入区快捷键：

- `Shift + Command + Space`：打开捕获面板
- `Enter`：保存输入，或复用当前选中的笔记
- `Shift + Enter`：插入换行
- `Tab`：在输入区和结果浏览之间切换
- `Down Arrow`：在输入为空时打开最近笔记
- `Escape`：执行返回、取消编辑、收起结果或关闭

结果区操作：

- `Up/Down Arrow`：在结果中移动
- `Cmd + C`：复制选中的笔记
- `Cmd + E`：编辑选中的笔记
- `Cmd + D`：删除选中的笔记
- `Cmd + P`：切换选中笔记的置顶状态
- `Cmd + Delete`：切换选中笔记的归档状态

## 结果浏览语义

当结果面板可见时：

- 选中一条笔记后按 `Enter`，会把它的内容拷贝进输入框，作为新草稿
- 编辑是通过主输入框原地完成的
- 置顶和归档都是切换操作
- 删除会立即删除当前选中的笔记

## 编辑模式

编辑已有笔记的方法：

1. 打开一个结果列表
2. 选中一条笔记
3. 按 `Cmd + E`
4. 在主输入框中修改内容
5. 按 `Enter` 保存，或按 `Esc` 取消

## 内置帮助

覆盖层内有两个自解释帮助入口：

- `/help`：查看可用斜杠命令
- `/keys`：查看可用键盘快捷键
